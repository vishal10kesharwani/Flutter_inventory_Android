# Inventory-Management-App
A Computer Vision based Inventory Management App using Flutter

---

# Requirements
- Git
- Flutter
  - Android Studio
  - VS Code
  - Visual Studio C++
- Firebase 

---

# Installation
- Git
```html
winget install --id Git.Git -e --source winget
```

- Heroku CLI
```html
npm install -g heroku
```

- Firebase-tools
```html
npm install -g firebase-tools
```
